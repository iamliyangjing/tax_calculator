# 工资税收检查APP规划

## 功能需求

### 1. 输入功能
- 月工资收入
- 公积金缴纳比例（个人部分）
- 社保缴纳基数和比例（或自动计算）
- 专项附加扣除（子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人）
- 其他扣除项

### 2. 计算功能
- 每月应扣税额计算
- 年度累计应纳税额计算
- 年度汇算清缴（退/补税额）

### 3. 展示功能
- 每月税收明细
- 年度税收汇总
- 退/补税金额

### 4. 辅助功能
- 税收计算规则说明
- 历史记录保存

## 技术栈
- React 18 + TypeScript
- Vite（构建工具）
- React Hook Form（表单处理）
- Zod（数据验证）
- Tailwind CSS（样式）

## 核心计算逻辑

### 1. 每月应纳税所得额
```
应纳税所得额 = 月度收入 - 5000元起征点 - 专项扣除（社保+公积金）- 专项附加扣除 - 依法确定的其他扣除
```

### 2. 累计预扣法计算每月税额
```
累计应纳税所得额 = 累计收入 - 累计免税收入 - 累计减除费用 - 累计专项扣除 - 累计专项附加扣除 - 累计依法确定的其他扣除
本期应预扣预缴税额 = (累计应预扣预缴应纳税所得额 × 预扣率 - 速算扣除数) - 累计减免税额 - 累计已预扣预缴税额
```

### 3. 年度汇算清缴
```
年度应纳税额 = (年度综合所得收入额 - 60000元 - 专项扣除 - 专项附加扣除 - 依法确定的其他扣除) × 适用税率 - 速算扣除数
退/补税额 = 年度应纳税额 - 已预缴税额
```

## 预期效果

用户可以输入个人工资信息、公积金比例等数据，系统将自动计算每月应扣税额，并根据年度累计情况计算年底可能的退/补税额，帮助用户验证税收扣除是否正确。
