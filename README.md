# 工资税收计算器

一个基于React + TypeScript的工资税收计算工具，帮助用户计算每月应扣税额及年度退/补税额。

## 功能特性

### 输入功能
- 月工资收入
- 其他月度收入（如交通补贴、餐补等）
- 公积金缴纳比例（个人）
- 社保缴纳方式
  - 按比例计算
  - 固定金额
- 专项附加扣除
  - 子女教育
  - 继续教育
  - 大病医疗
  - 住房贷款利息
  - 住房租金
  - 赡养老人
- 其他扣除项
- 年终奖
  - 单独计税
  - 并入综合所得计税

### 计算功能
- 每月应扣税额计算
- 年度累计应纳税额计算
- 年终奖税额计算
- 年度汇算清缴（退/补税额）

### 展示功能
- 当月税收情况
- 收入与扣除明细
- 年度税收汇总
- 年终奖税收情况
- 每月税收明细

## 技术栈

- **前端框架**: React 18
- **开发语言**: TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **表单处理**: React Hook Form
- **数据验证**: Zod

## 使用说明

1. 在输入表单中填写您的工资信息
2. 选择社保输入方式（比例/固定金额）并填写相应信息
3. 输入其他月度收入（如交通补贴等）
4. 填写专项附加扣除和其他扣除项
5. 输入年终奖金额并选择计税方式
6. 点击"计算税收"按钮查看结果

## 税收计算规则

### 每月应纳税所得额
```
应纳税所得额 = 月度总收入 - 起征点(5000元) - 社保扣除 - 公积金扣除 - 专项附加扣除 - 其他扣除
```

### 税率表（综合所得适用）

| 级数 | 应纳税所得额（全年） | 税率(%) | 速算扣除数(元) |
|------|----------------------|---------|----------------|
| 1    | 不超过36000元        | 3       | 0              |
| 2    | 超过36000元至144000元 | 10      | 2520           |
| 3    | 超过144000元至300000元 | 20     | 16920          |
| 4    | 超过300000元至420000元 | 25     | 31920          |
| 5    | 超过420000元至660000元 | 30     | 52920          |
| 6    | 超过660000元至960000元 | 35     | 85920          |
| 7    | 超过960000元          | 45      | 181920         |

### 年终奖计税方式

1. **单独计税**
   ```
   应纳税额 = 年终奖金额 × 适用税率 - 速算扣除数
   ```
   适用税率根据年终奖金额除以12个月后的金额确定

2. **并入综合所得计税**
   ```
   应纳税额 = (年度综合所得收入额 + 年终奖 - 60000元 - 专项扣除 - 专项附加扣除 - 其他扣除) × 适用税率 - 速算扣除数
   ```

## 注意事项

- 计算结果仅供参考，实际税额以税务部门核算为准
- 专项附加扣除需在个人所得税APP中如实申报
- 年度汇算清缴时，若已预缴税额大于年度应纳税额，可申请退税；若已预缴税额小于年度应纳税额，则需要补缴税款
